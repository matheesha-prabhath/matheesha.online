# Matheesha Prabhath — Portfolio

A modern minimalist portfolio website built for GitHub Pages.

## Pages
- `index.html` — Home / Hero
- `work.html` — Experience & Projects
- `creative.html` — Video editing, graphic design, motion work
- `about.html` — Bio, education, skills, achievements
- `contact.html` — Contact form + social links

## Deploying to GitHub Pages

1. Create a repo named `yourusername.github.io` (or any repo name for a project page)
2. Push all files to the `main` branch
3. Go to **Settings → Pages → Source → main branch / root**
4. Your site will be live at `https://yourusername.github.io`

## Customization Checklist

- [ ] Replace profile photo: Add your image to `/assets/` and update `about.html`
- [ ] Add CV file: Place `Matheesha_CV.pdf` in `/assets/` (the download button is ready)
- [ ] Add creative work: Replace placeholder gallery items in `creative.html` with real images or YouTube embeds
- [ ] Connect contact form: Sign up at https://formspree.io, add your form endpoint to the `<form action="">` in `contact.html`
- [ ] Update meta tags: Add your domain to `<link rel="canonical">` in each page's `<head>`
